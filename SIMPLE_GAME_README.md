Simple Godot Button Demo

1. Open the `death-dice` project in Godot (project.godot is in the repo root).
2. Open the scene: Scenes/Main.tscn
3. Press the "Play Scene" button (F6) or run the project. Click the "Press me" button — text will appear below it.
